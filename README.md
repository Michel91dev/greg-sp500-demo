# S&P 500 Streamlit Demo

Petite démo Streamlit pour visualiser l'indice S&P 500.

## Installation

```bash
# Créer un environnement virtuel
python3 -m venv .venv
source .venv/bin/activate

# Installer les dépendances
pip install -r requirements.txt
```

## Lancement

```bash
streamlit run streamlit_sp500_demo.py
```

## Fonctionnalités

- Affichage du prix actuel et variation
- Graphique d'évolution sur différentes périodes
- Volume de transactions
- Moyennes mobiles (20 et 50 jours)
- Statistiques principales

L'application s'ouvrira automatiquement dans votre navigateur à l'adresse http://localhost:8501
